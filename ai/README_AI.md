# AI Linear Model Project

## Project Purpose

This folder contains an AI-assisted implementation of Assignment 3: a simple
linear regression analysis that models **Salary** as a function of
**YearsExperience**. The project fits a straight line to the data, reports key
regression statistics, and produces annotated scatter plots in both **Python**
and **R**.

The deliverables include:

- Command-line scripts for batch analysis
- Jupyter notebooks for interactive, step-by-step analysis
- Saved plot images generated by the scripts

All work in this folder uses the shared conda environment **`regression-env`**.

---

## Dataset Description

**File:** `regression_data.csv`

| Column | Type | Description |
|--------|------|-------------|
| `YearsExperience` | numeric | Years of work experience |
| `Salary` | numeric | Annual salary |

The dataset contains **10 observations** with experience ranging from about 1.1 to
4.0 years and salaries from about 37,731 to 63,218. Both the scripts and
notebooks use this file to explore the relationship between experience and pay.

Example rows:

| YearsExperience | Salary |
|-----------------|--------|
| 1.1 | 39,343 |
| 2.0 | 43,525 |
| 4.0 | 63,218 |

## Regression Statistics Explained

### Slope

The **slope** is the change in predicted **Salary** for each one-unit increase in
**YearsExperience**. In this dataset, the slope is about **8285**, meaning each
additional year of experience is associated with roughly $8,285 higher predicted
salary.

It comes directly from the fitted model (`LinearRegression` in Python, `lm()` in R).

### Intercept

The **intercept** is the predicted salary when **YearsExperience** equals zero.
For this data, the intercept is about **29,204**. It represents where the
regression line crosses the y-axis.

Like the slope, it is taken from the fitted model coefficients.

### Pearson's r

**Pearson's correlation coefficient (r)** measures the strength and direction of
the **linear** relationship between the two variables. Values range from -1 to 1:

| r value | Interpretation |
|---------|----------------|
| near +1 | strong positive linear association |
| near 0 | little linear association |
| near -1 | strong negative linear association |

For this dataset, **r ≈ 0.886**, indicating a strong positive linear relationship
between experience and salary.

### MSE (Mean Squared Error)

**Mean Squared Error** measures average squared prediction error:

\[
\text{MSE} = \frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2
\]

where \(y_i\) is the observed salary and \(\hat{y}_i\) is the value predicted by
the fitted line. Lower MSE means the model's predictions are closer to the actual
values on average. For this dataset, **MSE ≈ 17,523,844**.

MSE is reported in squared salary units, so it is most useful for comparing models
on the same dataset rather than interpreting on its own.

## Output Plots

Running the scripts creates two annotated scatter plots:

| File | Created by |
|------|------------|
| `regression_plot_python.png` | `linear_model.py` |
| `regression_plot_r.png` | `linear_model.R` |

Each plot includes:

- **Observed data points** (steelblue) showing actual salary vs. experience
- **Regression line** (red) showing the fitted linear model
- **Text annotation** with the regression equation and key metrics (Pearson's r,
  MSE, and R²)
- **Axis labels and title** identifying the variables
- **Legend** on the Python plot (matplotlib); R uses matching ggplot2 styling
  in both `linear_model.R` and `linear_model_r.ipynb`

The plots visualize the same fitted model reported in the terminal output. Python
uses matplotlib; R uses ggplot2 in both the script and notebook so the saved PNG
and notebook display match.

## Running the Notebooks

Interactive:

```bash
jupyter lab --no-browser --port=2001
```

| Notebook | Kernel |
|----------|--------|
| `linear_model_python.ipynb` | Python 3 |
| `linear_model_r.ipynb` | `R (regression-env)` |

Execute from the command line:

```bash
jupyter nbconvert --to notebook --execute linear_model_python.ipynb --inplace

jupyter nbconvert --to notebook --execute linear_model_r.ipynb \
  --inplace --ExecutePreprocessor.kernel_name=ir_regression_env
```

Each notebook loads the data, explores the relationship, fits the model, shows
predicted values with residuals, prints slope/intercept/Pearson's r/MSE, and
displays an annotated regression plot.
